### js数据类型
**基本类型**：
数字类型number
字符串类型string
布尔类型boolean
BigInt
null
undefined
**引用类型**：
对象类型object
### typeof
判断数据类型
```js
typeof undefined // 'undefined'
typeof null  // 'object'
typeof 222 // 'number'
typeof 'aaa' // 'string'
typeof true // 'boolean'
typeof 123n // 'bigint'
typeof Symbol() // 'symbol'
typeof function() {} // 'function'
typeof {name: 'cpp'} // 'object'
```
